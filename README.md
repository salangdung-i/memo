## 프로젝트 소개 
텍스트를 입력후 ➕ 버튼 또는 키보드에서 Enter를 누르면 메모장에 텍스트가 입력됩니다.<br>
🗑 버튼을 클릭하면 해당 메모가 삭제됩니다.
