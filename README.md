## 바닐라JS를 이용해 만드는 Memo📝
<img src="img/memo-main.png" title="사용 화면" width="300px">   

##### 뭐든 메모할 수 있는 간단한 Memo 입니다.    
<br>


## 프로젝트 소개 
- 텍스트를 입력후 ➕ 버튼 또는 키보드에서 Enter를 누르면 메모장에 텍스트가 입력됩니다.
- 🗑 버튼을 클릭하면 해당 메모가 삭제됩니다.  
<br>

## 기술스택 
- HTML, CSS, JAVASCRIPT  
<br>
